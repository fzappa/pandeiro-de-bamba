## Samples do pandeiro

O app procura os samples desta pasta pelos nomes abaixo:

- `P.wav`
- `P_mut.wav`
- `Pm.wav`
- `D.wav`
- `D_mut.wav`
- `Dm.wav`
- `T.wav`
- `H.wav`
- `F.wav`
- `R.wav`

Voce pode trocar qualquer arquivo mantendo o mesmo nome. Use WAV, mono ou stereo. Se algum arquivo
nao carregar, o app usa o som sintetizado interno como fallback.
